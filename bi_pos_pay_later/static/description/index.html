Odoo POS Pay Later Odoo apps

For Point of sales Odoo POS Many customers, search more and more convenient ways of payment in odoo. Through this Odoo apps, we are giving chance to customers to pay partially in POS or delay payment on thier account. Users can see previous orders such as draft orders, paid orders, posted orders or reorder, invoiced orders, make payments for partially paid or unpaid orders and also create draft orders without any payment. this can be useful feature for Odoo POS.After installing this Odoo module POS user can also able to see all orders on screen in POS and reorder that same order and reprint it. The user can takes products on 100% Credit, and will pay later means user doesn't pay anything i.e 0.

Spanish Translation Available.

Features

Create Draft POS Order.

User can create a draft pos order without any payment.

Partially Pay for Orders.

User can allow customers to pay partially for their pos orders.

Filters for POS Orders.

User can apply filters on pos orders to see paid, unpaid, posted and invoiced orders.

Receipt with Due and Total Paid Amount.

User can print receipt with due and total paid amount.

Generate Picking for Draft/Partial Order.

Picking order will generated for draft or partial paid order.

Pay Partial Payment Multiple Time.

Now user can pay partial order amount multiple time.
