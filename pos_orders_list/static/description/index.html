All POS Orders List Odoo Apps

This Odoo apps help to see the list of all the orders in Point of sales, user can also see individual customers list of orders.After installing this Odoo module POS user can also able to see all orders on screen in POS. .

Spanish Translation Available.

Features

Allow User to Configure the Limit of POS Order.

Customer can manage the pos orders as per the configuration.

Allow user to see Different Orders from Different Session.

Customer can see the pos orders which are paid, on draft or posted.

User can See POS Orders.

User can see pos orders as per the configuration of the customer in pos screen.

Configuration for Barcode.

User can see barcode in pos receipt as well as in order.

Installation Guide

Before installing this module you have to install following packages in your system.(If you have then Ignore this)

sudo pip3 install python-barcode

OR

sudo pip install python-barcode

Load POS Order Configuration

User have to select from options from the settings to set limit to load pos orders.

Only Current Session

Pos order for that current session can be seen by user by selecting this option.

In POS Screen "Orders" button appear,user can see all POS orders directly from POS.

When user clicks on "Orders" button he/she can see pos order for that current session.

Only Current Day Orders

POS order for the current day can be seen by user by selecting this option.

In POS Screen "Orders" button appear,user can see all POS orders directly from POS.

When user clicks on "Orders" button he/she can see pos order for that current session.

Load Last 3 Session's Order

POS order for the last 3 sessions can be seen by user by selecting this option.

In POS Screen "Orders" button appear,user can see all POS orders directly from POS.

When user clicks on "Orders" button he/she can see pos order for the last 3 sessions.

Load Last 5 Session's Order

POS order for the last 5 sessions can be seen by user by selecting this option.

In POS Screen "Orders" button appear,user can see all POS orders directly from POS.

When user clicks on "Orders" button he/she can see pos order for the last 5 sessions.

Load All Session's Order

Pos order for all sessions can be seen by user by selecting this option.

In POS Screen "Orders" button appear,user can see all POS orders directly from POS.

When user clicks on "Orders" button he/she can see pos order for all session.

Load Draft & Posted Orders

User have to enable "Show draft Orders" and "Show Posted Orders".

In POS Screen "Orders" button appear,user can see all Pos order which are in draft state and posted orders can be seen.

Draft Orders

In POS Screen "Orders" button, user can see all POS order which are in "Draft" state.

Posted Orders

In POS Screen "Orders" button, user can see all POS order which are in "Posted" state.

Configuration for Barcode

Barcode in POS Receipt

Barcode and order reference in pos order in the back-end.
