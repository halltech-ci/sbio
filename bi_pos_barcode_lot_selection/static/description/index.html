POS Automatic lot selection with barcode Odoo Apps

Point of sale lot selection using barcode Scanner Odoo Apps

POS lot selection with Barcode Odoo app helps to create unique barcode-lot/serial number combination name , which will useful to print product labels based on lots/serial number, Users can scan that on pos to add a product along with lot. This module allows you to select valid lot/serial number from selection field in popup. We have also added validations to restrict user by adding wrong/not valid lot/serial number. Users can also configure lot selection based on Operation type location or from all locations (configurable option).

Features

Print Product for Barcode-Lot/Serial Combination Name.

User can print product based on barcode-lot/serial number.

Scan & Add Product in POS for Specific LOT.

User can scan barcode-lot/serial number in pos in order to add product with specific lot/serial number.

Selection of Valid LOTs.

We have added new Selection field for valid lot/serial number in Lot/serial add popup.

Quantity with Respect to Lot/Serial Number.

User can see available quantity of particular lot/serial number in selection field in Popup.

Lot/Serial Configuration for Location.

User can configure lot/serial number selection based on i) All Locations or ii) Operation Type Location from pos config.

Lot/Serial Number in Receipt.

In receipt user can see used lot/serial number and also expiration date.

Auto Remove Product When Lot/Serial Number Expired

When product lot/serial number has expired then auto remove product from point of sale.

Auto Disable Product Once Its Out of Stock

User cannot add product into cart which has out of stock.

Works in Online & Offline both.

This module will works on online and offline both in POS.

Validations.

We have added validations to restrict user by adding wrong/not valid lot/serial number.

POS Configurations

Under Point of Sale user need to configure "Select Lots from", If user select "All Locations" option then lot numbers for product will appear from all locations, user can also change default stock location from operation type.

Product Barcode Lot Number

User can see available LOT numbers of product.

User can give expiry date for particular lot number.

User have to remove the product when Lot/Serial Number expired.

User can also see available onhand quantity of products with locations and lot number.

User can also see generated barcode lot with combination of product barcode and lot number, user can also print this number from "Product Barcode LOT (PDF)" menu.

Product Barcode LOT (PDF) Report

On scaning product barcode lot number user can see product added to point of sale.

On pop-up user can see all lot numbers of product from all location with available quantity.

Also user can see expired lot number of product is auto remove from POS pop-up.

User can also enter lot number.

POS Order Receipt

User can see point of sale order receipt with product barcode.

POS Order

User can see selected lot number on order and also stock location.

Once product is out of stock it is disable and user can not add product into cart.
